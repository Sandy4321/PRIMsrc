PRIMsrc

Bump Hunting by Patient Rule Induction Method for Survival, Regression and Classification

View on GitHub Download .zip Download .tar.gz

Idea

PRIMsrc is a generic method to perform a Bump Hunting search in high-dimensional data. The method generates decision rules delineating a region in the multi-dimensional predictor space, where a target function of interest assumes larger (or smaller) values than its average over the entire space.

Formal Goal - Description

Consider a supervised setting, where the target function of interest is a univariate response vector (discrete, continuous or possibly censored) of a multivariate vector of input explanatory variables (continuous or discrete).

Denote by R^{p} a p-dimensional Euclidean space and f(x) the unknown target function of interest of input explanatory variables x=(x_{1}, x_{2}, … , x_{p}) of support S \subseteq R^{p}.

The goal is three fold :

  • Characterize a sub-region R \subseteq S in the input space over which the target function f(x) is expected to have a local maximum (≠ noise). The region R could be any smooth shape (e.g. a convex hull), possibly disjointed
  • Determine which of the input variables (x_{1}, x_{2}, … , x_{p}) are important/relevant in the process
  • Identify corresponding sub-groups of samples within region R

We chose to achieve these goals simultaneously by using a Bump Hunting approach. PRIMsrc implements a unified treatment of non-parametric Bump Hunting in Survival, Regression and Classification settings (SRC) using an algorithm derived from the Patient Rule Induction Method (PRIM), initially introduced by Fisher & Friedman in 1999. (See publications list on the Wiki page)-Publications). PRIMsrc estimates the target region R of interest where the response is larger (or smaller) than its average over the entire space. This is done by generating decision rules delineating an approximation of the region R, which is shaped in PRIMsrc as a hyper-dimensional box or hyper-rectangle, possibly disjointed.

Assumptions

Assumptions are that the multivariate input variables can be discrete or continuous and the univariate response variable can be discrete (Classification), continuous (Regression) or a time-to event, possibly censored (Survival). It is intended to handle low and high-dimensional multivariate datasets, including the situation where the number of covariates p exceeds or dominates the number n of samples (p > n or p >> n paradigm).

Why Use PRIMsrc?

The problem of "Bump Hunting" covers mathematical / statistical tasks such as:

  • Mode(s) Hunting
  • Local/Global Extremum(a) Finding
  • Subgroup(s) Identification
  • Outlier(s) Detection
  • …

Direct applications are multiple, such as:

  • characterization and stratification of patient groups with respect to diagnostic and/or prognostic purposes with benefits in personalized and economical medicine
  • system reliability analysis in engineering
  • duration analysis/modeling in economics
  • event history analysis in sociology
  • financial securities return
  • insurance risk assessment
  • …
